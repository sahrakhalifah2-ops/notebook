# Research Notebook

A record of thought, branches, critique, and revision.

## Structure

- `raw/` — messy drafts, fragments, unfinished
- `journal/` — cleaned entries, dated, readable
- `posts/` — public pieces

## What this is

Not polished essays. Not content. A working log of how thought actually moves.

The point is process: the branch, the rejection, the correction, the landing.
